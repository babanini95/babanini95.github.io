[**Battle of Guardians**](https://store.steampowered.com/app/2286560/Battle_of_Guardians/) is a real-time online multiplayer PVP and PVE free-to-play fighting game developed by my previous company, [**Miracle Gates Entertainment**](https://www.miraclegates.com/). I was part of the development team for this project. I contributed to UI implementation and performance optimization, as well as porting the game to Android.
<!--more-->

To port the game to Android, we need to minimize the game size and optimize the performance to run smoothly on mobile devices. To do that, We customized [Locus Bundle](https://github.com/locus84/Locus-Bundle-System), a Unity Addressable alternative, to manage our game assets.
